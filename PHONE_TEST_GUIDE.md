# Facelympic 휴대폰 테스트

## 접속 주소

- 일반 플레이: `https://facelympic.vercel.app`
- 기술 진단: `https://facelympic.vercel.app/?debug=1`

Pi Browser와 기본 모바일 브라우저에서 같은 HTTPS 주소를 사용합니다.

## 한 번의 테스트 순서

1. 카메라 권한을 허용하고 단거리 종목을 선택합니다.
2. 무표정, 웃음, 입 벌리기, 입 오므리기, 눈썹, 고개 좌우 기울이기를 차례로 보정합니다.
3. 진단 HUD의 얼굴 인식이 유지되고 인식 FPS가 20 이상인지 확인합니다.
4. 웃음값이 속도에 따라 연속 변화하고 고개 기울기 값이 좌우에서 서로 다른 부호인지 확인합니다.
5. 한 경기를 완주한 뒤 결과 화면과 오늘의 세계 순위가 표시되는지 확인합니다.
6. 종목 선택 화면으로 돌아가 오늘 세계 1등 기록이 표시되는지 확인합니다.
7. 소리를 켠 채 다시 달려 카메라·게임·오디오가 함께 유지되는지 확인합니다.

|---|---|---|---:|---|---|---|---|
|  |  |  |  |  |  |  |  |

문제가 생기면 기기, OS, 브라우저 버전, 발생 단계와 화면의 진단값을 함께 기록합니다. 얼굴 영상은 저장하거나 전송하지 않습니다.
